# Virtual-AI-voice-assistant
An AI-powered voice assistant built with Python that can execute commands, open applications/websites, and answer queries using APIs.

🚀 Features
🎤 Voice command input
🌐 Open websites & applications
📰 Fetch latest news
🤖 AI-based responses (via API)
🔊 Text-to-speech output
🛠 Tech Stack
Python
SpeechRecognition
pyttsx3
External APIs (for AI & news)
📦 Setup
git clone https://github.com/your-username/voice-assistant.git
cd voice-assistant
pip install -r requirements.txt
▶️ Run
python main.py
🔐 Configuration

Create a .env file for API keys:

API_KEY=your_api_key_here
📌 Notes
Ensure microphone access is enabled
Keep API keys private (do not upload .env)
